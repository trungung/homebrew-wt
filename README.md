# Trungung Wt

## How do I install these formulae?

`brew install trungung/wt/<formula>`

Or `brew tap trungung/wt` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "trungung/wt"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
